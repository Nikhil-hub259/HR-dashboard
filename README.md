This project is an interactive HR Analytics Dashboard built using Tableau to analyze employee data and provide insights into workforce trends, attrition patterns, employee demographics, and job satisfaction.
The dashboard helps HR teams and business stakeholders make data-driven decisions by identifying key factors affecting employee retention and workforce performance.

Project Objectives:
Analyze employee attrition trends
Understand workforce demographics
Identify department-wise attrition patterns
Evaluate employee satisfaction levels
Provide visual insights for HR decision-making

The Tableau dashboard includes the following visualizations:
KPI Overview
Total Employees
Attrition Count
Attrition Rate
Active Employees
Average Age
Attrition Analysis
Attrition by Education
Attrition by Gender
Attrition Rate by Gender & Age Group
Department-wise Attrition
Employee Distribution by Age Group
Gender Distribution
Job Satisfaction Ratings across departments and roles
